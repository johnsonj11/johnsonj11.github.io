R Practical 0: Cherry Tree Measurements Analysis

(Recap with R and R Markdown)

Introduction

The aim of this practical is to provide the opportunity for you to review the basics of using ‘R’, review the basics of using ‘R Markdown’ for producing reports and to provide a clear example of the expected report format required for the R-Practicals in this course, three of which are assessed and count towards your final assessment for the module.

In this practical we will explore the ‘cherry trees’ data set. This data set contains measurements of the height (in ft), volume (in cubic ft) and diameter (in inches; measured at 4ft 6in above the ground) of timber in 31 felled black cherry trees. This data comes from a built-in data frame in R called trees, however, we will read the data in from a file in this practical.

You are asked to complete the following tasks. You will be guided through tasks 1 to 7 at the start of the tutorial, and will complete tasks 8 to 10 independently in the remaining tutorial time.
